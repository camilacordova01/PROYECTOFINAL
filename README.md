# PROYECTO-FINAL-CORDOVA-MORON

Este repositorio contiene todas las actividades desarrolladas durante el curso de Sistematización y métodos estadisticos, como parte de mi proceso de aprendizaje en RStudio.

Incluye:

-Scripts en R utilizados para la resolución de ejercicios.
-Conjuntos de datos empleados en cada actividad.
-Documentación y archivos de apoyo del proyecto final.
-Resultados obtenidos de los análisis realizados.

El objetivo principal de este repositorio es mostrar mi progreso, organización y dominio de las herramientas de análisis de datos con R.
